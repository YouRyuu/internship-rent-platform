### 接口设计
- 房屋
    - 根据筛选条件查询房屋(list)
        - 返回
            - 房屋id
            - 城市
            - 区
            - 地点（小区/location）
            - 价格
            - 名称
    - 根据房屋id查询房屋
        - 返回
            - 房屋所有信息
            - 房东信息
            - 评价
            - 房屋图片
    - CUD
- 用户
    - 创建
    - 更新
    - 查找
    - 删除
- 订单
    - CURD
- 留言
    - CURD
- 评价
    - CURD
